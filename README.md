# Traffic-Simulator
Prerequisiti:

1) Installare g++ 4.9:

    sudo add-apt-repository ppa:ubuntu-toolchain-r/test
    sudo apt-get update
    sudo apt-get install g++-4.9

2) Installare gtkmm:

    sudo apt-get install libgtkmm-3.0-dev 

3) Installare glade

    sudo apt-get install glade
    
4) Libreria SFML: http://www.sfml-dev.org/download/sfml/2.3.2/
